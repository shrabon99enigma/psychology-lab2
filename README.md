# 🕵️ FBI Mobile Habit Scanner

A fun and interactive Streamlit web app that scans your smartphone habits in a humorous FBI-style experience.

---

## 🚀 Live Demo

👉 https://fbi-mobile-habit-scanner.streamlit.app

---

## ✨ Features

- 🔍 FBI Style Mobile Scan
- 🎲 Random Mobile Habit Detection
- 📊 Mobile Addiction Score
- 🧠 Funny Verdict System
- 🎈 Celebration Animation
- 🔄 Scan Again Button
- ❤️ Beginner Friendly UI

---

## 🛠 Built With

- Python
- Streamlit

---

## ▶️ Run Locally

```bash
git clone https://github.com/shrabon99enigma/psychology-lab2.git

cd psychology-lab2

pip install -r requirements.txt

streamlit run psychology.py
```
